Adding a new entry
Instructions for how to add a new entry to the library

Amazing!! A whole new entry for the Library! Thank you for having the interest and taking the time to contribute to the DIYbiosphere project!!

The process for adding a new entry is just 3 small steps: first, fill in the following form with the data and description of your initiative. Then, a text file will be generated for you entry which you have to save to your drive. Finally, you have to upload the file to the repository in the collection folder, some additional steps apply depending on whether you are a non-member, member, or team member.
